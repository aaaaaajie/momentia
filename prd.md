一、产品核心定位
面向自媒体创作者、手账爱好者、学生的轻量工具类产品，用户上传图片/输入文字描述/粘贴语音转文字内容，工具会自动提取关键元素，通过多模态大模型生成风格统一的创意拼贴画（如复古手账风、赛博朋克风、治愈插画风），支持一键导出高清图和拼贴素材包，主打“零设计基础也能快速做创意视觉内容”。

二、核心功能（开发难度低，无硬件依赖）
多模态输入
◦ 文字输入：直接写拼贴主题（如“海边日落手账”）；
◦ 图片上传：导入1-3张素材图，模型提取色彩/元素特征；

智能拼贴生成（仅用 OpenAI，无需 Stable Diffusion）
◦ 调用 OpenAI 多模态模型（如 GPT-4o）解析用户输入（文字+图片），生成“拼贴计划”（布局 JSON + 风格描述 + 配色方案 + 图层清单 + 文案建议）；
◦ 可选：调用 OpenAI 图像生成能力生成拼贴所需的背景、纸张纹理、贴纸/小图标等装饰元素（统一风格与色调）；
◦ 内置 5 种主流拼贴风格模板（枚举），支持在模板参数上做轻量自定义：主色/辅色、留白、元素密度、字体风格、元素大小。

输出与导出
◦ 生成高清成品图（PNG/JPG，支持 2x/4x 分辨率）；
◦ 导出拼贴素材包（ZIP）：背景、装饰元素、用户照片处理后的裁切/抠图结果、布局 JSON（便于复用与二次编辑）。

实现要点（工程化约束，提升稳定性）
◦ 约束模型输出为严格 JSON Schema（canvas/layers/assets/text），避免自由发挥导致无法合成；
◦ 服务端合成：按布局 JSON 进行图层渲染与排版，做边界/重叠校验与自动缩放兜底；
◦ 失败降级：当装饰元素生成失败时，回退到“纯模板 + 用户照片 + 文案”的可用结果。

限制说明（真实预期管理）
◦ 仅用 OpenAI 也可闭环，但成本与生成速度与图片尺寸强相关；
◦ 对“极一致的风格/角色一致性”能力有限，建议优先做“手账/拼贴质感”方向（纸张、贴纸、纹理）以保证稳定体验。
